pip installs Python packages. It knows about various source control systems and can install direct from a git repository. For example, to install Tornado HEAD from GitHub:

$$code(lang=console) sudo pip install git+https://github.com/facebook/tornado.git $$/code

Uninstall is easy too and seems to be based on simple pattern matching.

$$code(lang=console) sudo pip uninstall tornado $$/code
